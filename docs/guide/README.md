## 下载安装

Xray支持各种平台,并且您可以从多种渠道和方式获得Xray的各种版本.  
  
请点击[如何下载安装Xray](./install)以获取Xray

下载并安装Xray后，只需对他进行配置即可使用.  
  
请点击[如何配置运行Xray](./config)以学习最简单的配置方式.

## 命令参数

Xray有多种命令和参数可用,因此变得灵活和强大.   
  
请点击[Xray的命令参数](./command)查看Xray的更多命令和参数用法.
